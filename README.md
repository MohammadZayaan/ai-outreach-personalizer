# AI Outreach Personalizer

An AI-powered cold outreach personalization tool built with Next.js, TypeScript, and Tailwind CSS.

Users can upload CSV lead data, generate personalized outreach messages, and export generated results.

## Live Demo

https://ai-outreach-personalizer.vercel.app/

---

## Features

- CSV Upload
- CSV Parsing
- Dynamic Leads Table
- AI Outreach Message Generation
- Export Generated Messages as CSV
- Modern SaaS-style UI
- Toast Notifications
- Responsive Design

---

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- React
- Vercel Deployment
